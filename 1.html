HTML이란 무엇인가?

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications.

HTML is a command that defines the font size, font color, graphic shape, and document movement (hyperlink) of a document and is used to create a website. The command used in HTML is called a tag, which is represented by the angle brackets "<>". In general, a tag consists of two pairs representing the beginning and the end, but in some cases, such as tags such as "img" and "br", the end tag begins with a slash character [/]. These HTML-written documents are called HTML documents, and the web browser interprets them and shows them to users. In HTML, a document can recognize a separate code to create a complete hypertext, as well as link words or paragraphs to other places or files on the Internet.

HTML was created to define the format of electronic documents and defined as a subset of the international standard SGML. HTML is a language created in SGML, particularly by highlighting hypertext, and is composed of plain text in ASCII codes. The language does not require a separate compiler and is recognized as an easy-to-use language that can be interpreted by web browsers.
